# App Overview

FYNX Finance World is a trading + investing operating system.

## Core modules
- **Dashboard** — overview, stats, quick actions
- **Trader Journal** — log trades, track performance, patterns
- **Calculators** — risk, position sizing, ATR tools
- **News** — market headlines and key events
- **Calendar** — macro events, reminders, sessions
- **Profile & Settings** — account, preferences, security

## Design principle
Fast, clean, minimal. Built for daily use.
